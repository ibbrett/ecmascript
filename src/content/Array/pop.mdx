- to remove item:
  - from index of array use [splice()](../splice-remove-item/)
  - from the beginning of array use [shift()](../shift/)

<div class="copy-to-clipboard-wrap" id="code-sample-01">
<CopyToClipboard id="code-sample-01"></CopyToClipboard>
```js
const list = ['apple', 'banana', 'cantaloupe'];
console.log('initial list', list);
// Expected output: ['apple', 'banana', 'cantaloupe']

// get/remove first item
const returnValue = list.pop();

console.log('get/remove first item', list);
// Expected output: ['apple', 'banana']

console.log('removed item', returnValue);
// Expected output: cantaloupe
```
