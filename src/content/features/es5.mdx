- "strict mode"
- JSON support
- String.prototype.trim()
- String.prototype.charAt() to return a single character from a given position in a string.
- new Array static (class) method: 
  - Array.isArray(): determines whether the passed value is an Array

- A comma after the final pair of values in an object no longer causes a syntax error

```js
// prior to ES5, this trailing comma caused an error
const example = { "property1":"value1", "property2":"value2", };
```
