{frontmatter.what}  

flattens an array to a specified depth, meaning that all subarray elements (up to the specified depth) are concatenated recursively

<div class="copy-to-clipboard-wrap" id="code-sample-01">
<CopyToClipboard id="code-sample-01"></CopyToClipboard>
```js
const list = ['apple', ['cherry', ['pumpkin', ['pecan', ['blueberry', 'peach']]]]];
console.log(list);
// expected output: ['apple', Array(2)]

console.log(list.flat()); // defaults to 1, same as .flat(1)
// expected output: ['apple', 'cherry', Array(2)]

console.log(list.flat(2));
// expected output: ['apple', 'cherry', 'pumpkin', Array(2)]

console.log(list.flat(Infinity));
// expected output: Array ['apple', 'cherry', 'pumpkin', 'pecan', 'blueberry', 'peach']
```
