{frontmatter.what}  

- [splice()](../splice/) is mutable - it changes the original array
- toSpliced() is immutable - it does not change the original array

why use toSpliced instead of splice:
- This allows you to mutate a copy of an array without needing to clone the array first

<div class="copy-to-clipboard-wrap" id="code-sample-01">
<CopyToClipboard id="code-sample-01"></CopyToClipboard>
```js
const list = ['orange', 'cantaloupe', 'dragon fruit', 'apple', 'lemon', 'banana'];

// The toSpliced() method is the copying version of the splice() method
// Inserting an element at index 1
const add_item_to_list = list.toSpliced(1, 0, "Pineapple");
console.log(add_item_to_list);
// Expected output: ['orange', 'Pineapple', 'cantaloupe', 'dragon fruit', 'apple', 'lemon', 'banana']

// Deleting two elements starting from index 2
const remove_item_from_list = list.toSpliced(2, 2);
console.log(remove_item_from_list);
// Expected output: ['orange', 'cantaloupe', 'lemon', 'banana']

// Replacing one element at index 1 with two new elements
const alter_list = list.toSpliced(1, 1, "grapes", "raspberries");
console.log(alter_list);
// Expected output: ['orange', 'grapes', 'raspberries', 'dragon fruit', 'apple', 'lemon', 'banana']
```

```js
const months = ['Jan', 'March', 'April', 'June'];

// Inserts at index 1
let splicedMonths = months.toSpliced(1, 0, 'Feb');
console.log(splicedMonths);
// Expected output: Array ['Jan', 'Feb', 'March', 'April', 'June']

// Replaces 1 element at index 4
splicedMonths = months.toSpliced(3, 1, 'May');
console.log(splicedMonths);
// Expected output: ['Jan', 'March', 'April', 'May']

console.log(months);
// Expected output: Array ['Jan', 'March', 'April', 'June']
```
