{frontmatter.what}

- overrides Object toString method 
- calls [join()](../../Array/join/) internally, returning a comma-delimited string
- similar to [toLocaleString()](../../Array/tolocalestring/)

<div class="copy-to-clipboard-wrap" id="code-sample-01">
<CopyToClipboard id="code-sample-01"></CopyToClipboard>
```js
const list = [1, 3, 5, 7, 9];

console.log('list', list);
// Expected Output: [1, 3, 5, 7, 9]

console.log('list as a string', list.toString());
// Expected Output: 1,3,5,7,9
```
