{frontmatter.what}

- reverse() is mutable - it changes the original array
- [toReversed()](../toreversed/) is immutable - it does not change the original array

<div class="copy-to-clipboard-wrap" id="code-sample-01">
<CopyToClipboard id="code-sample-01"></CopyToClipboard>
```js
const list = [ 1, 5, 3, 7, 9 ];
console.log('unsorted list', list);
// Expected Output: [ 1, 5, 3, 7, 9 ]

list.reverse();
console.log('reverse list order', list);
// Expected Output: [9, 7, 3, 5, 1]
```
